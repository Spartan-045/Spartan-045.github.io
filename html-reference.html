HTML Reference Page

What Makes a Valid HTML Page?

The <html>, <DOCTYPE __>, <heading>, and <body> lines of code.

HTML Element

Description

The HTML element in (normally) in the beginning of the code is to define the file as a html file.

Syntax

The syntax is <html> and </html>.

Example

<html>This is how you do it.</html>

Heading Element

Description

The heading element in an HTML page is quite important to the main website code, as it implies all the font changes, paragraph breaks before and after, and any white space necessary to render the heading. There are up to 6 different heading elements, ranging from h1 to h6 with h1 being the most important and h6 being the least.

Syntax

The syntax of the Heading Element is (<heading> and </heading>) or (<head> and </head>).

Example

<head> This is how to do it: </head>

Title Element

Description

This element is used to name the webpage that you are working.

Syntax

<title> and </title>.

Body Element

Description

The body element in the lines of code is used to contain the actual content of the page.

Syntax

The syntax is: <body> and </body>

Example

<head>This is how to do it.</head>

Paragraph Element

Description

The paragraph element is used to write the main content in the page. Although it is normally in a "standard text", you can adjust it with a variety of different code.

Syntax

The syntax is: <p> and </p>.

Example

<p> This is how the element works!</p>
